-------------------------------------------
クライアント研修（ビリヤード）
-------------------------------------------

開発環境：Visual Studio 2015
開発言語：C++

DirectXTKを使用
Billiard/Release フォルダ内のexeファイルから実行できます

■ゲームルール
　ルール : ナインボールのみ
　プレイ人数 : 一人(CPU等も未実装)
　手球を打って9番(黄色)の的玉がポケットしたらゲーム終了(画面左上にFinishd!と表示される)
　ファウルは未実装(手球がポケットしたときは初期位置に移動)

■操作方法(キーボード)
　・十字キー(左右) : 視点の回転
　・スペースキー  : 手球を打つ（カメラ正面方向）

　※デバッグ用
　　・「W」「A」「S」「D」キー : 手球の移動
　　・エスケープキー : 手球を初期位置に移動

■こだわった点
　・衝突判定関係（http://marupeke296.com/COL_main.htmlからかなり引用しています）

■反省点
　・C++での開発が初めてだったのとDirectXや衝突判定をできるだけライブラリを使わないようにした結果、完成させられなかった。
　・早い段階でライブラリを使用するかどうかの判断ができるように時間管理をするべきでした。

■実装できなかった機能
　・タイトル、リザルトのシーンとその遷移
　・音関係
　・エフェクトや影、玉のテクスチャ等
　・UI関係
